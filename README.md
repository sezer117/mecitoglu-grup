# Mecitoğlu Grup Catering Web Sitesi

## Bilgisayarda açma
`index.html` dosyasını çift tıklayın veya VS Code'da Live Server ile açın.

## Vercel'e yükleme
1. Bu klasörün içindeki tüm dosyaları GitHub'daki `mecitoglu-grup` deposuna yükleyin.
2. Vercel'de **New Project** seçin.
3. GitHub deposunu seçin.
4. Framework Preset: **Other**.
5. Build Command ve Output Directory alanlarını boş bırakın.
6. **Deploy** düğmesine basın.

## Düzenleme
- Telefon, e-posta ve adres: `index.html` içindeki İletişim bölümünde.
- Fotoğraflar: `assets/images` klasöründe.
- Renkler ve görünüm: `style.css` dosyasında.
- Belgeler: `index.html` içindeki `belgeler` bölümüne PDF bağlantıları eklenebilir.
